## 0.1.0

- Initial release
- Event recording with session scoping
- JSONL file-based storage
- Retention policies (standard, conservative, aggressive, none)
- Survey system with three response types (options, text, combined)
- Event-based triggers with property matching
- Scheduled/delayed survey triggers
- Attempt tracking and cooldowns
- Debug configuration for testing
- Light/dark mode support in survey UI
- Remote config loading with auto-refresh
